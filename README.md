# Pokedex
포켓몬 도감 앱 과제

## 개발 환경
- Xcode 16.2
- macOS 15.2
- Deployment Target iOS 16.6

## 기술 스택
- UIKit, ComposableLayout, RxSwift
- MVVM

## 주요 기능
- Pokemon API를 활용, 포켓몬 목록을 보여주는 컬렉션 뷰
- 무한 스크롤
